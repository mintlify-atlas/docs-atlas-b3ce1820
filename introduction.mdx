# Welcome to Better Hub

Better Hub reimagines code collaboration on top of GitHub. Built by the team at Better Auth, it delivers a faster, cleaner experience for everything from repo overviews to PR reviews and AI-powered assistance.

## Why Better Hub?

We spend a lot of our time on GitHub. So we decided to build the experience we actually wanted — faster navigation, cleaner UI, and AI that actually helps.

## Key Features

### Repo Overview
Cleaner layout with rendered README, interactive file tree, and activity feed.

### PR Reviews
Inline diffs, AI-powered summaries, and streamlined review comments.

### Issue Management
Triage, filter, and act on issues faster with intelligent organization.

### Ghost AI Assistant
Review PRs, navigate code, triage issues, and write commit messages. Toggle with `⌘I`.

### Command Center
Search repos, switch themes, navigate anywhere with `⌘K`.

### CI/CD Status
View workflow runs and compare across branches.

### Security Advisories
Track vulnerabilities per repo in one place.

### Keyboard-First
Most actions accessible via shortcuts for maximum efficiency.

### Browser Extension
Adds "Open in Better Hub" button on GitHub pages (Chrome & Firefox supported).
